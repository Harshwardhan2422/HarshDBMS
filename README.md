📊 HarshDBMS (Customized Database Management System)

🛠️ Technology

- Java Programming

---

📌 Description

1. This project is used to illustrate the concept of a Database Management System (DBMS).

2. In this project, we create a virtual table to store and manage employee records.

3. We implement our own logic to perform operations similar to SQL queries.

4. Based on user input, the system executes appropriate operations like Insert, Delete, Update, and Select.

5. This project is implemented using the Collection Framework of Java (LinkedList).

---

🚀 Features

- Insert Record
- Display All Records
- Search by ID and Name
- Delete by ID and Name
- Update Name and Address
- Aggregate Functions:
  - Count
  - Sum
  - Average
  - Maximum
  - Minimum

---

▶️ How to Run

javac CDMS.java
java CDMS

---

💡 Learning Outcomes

- Understanding of DBMS concepts
- Implementation of CRUD operations
- Use of Java Collections
- Menu-driven programming
